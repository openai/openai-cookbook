## Semantic Similarity Search with Oracle Database 23ai

This repository contains a **cloud-first, reproducible cookbook** demonstrating
how to build **semantic similarity search** applications using **Oracle Database 23ai**
native **VECTOR** support and **OpenAI embeddings**.

The cookbook shows how embeddings can be stored **directly inside Oracle Database**,
queried using **SQL-native vector distance functions**, and exposed through both
a **Jupyter Notebook** and an **interactive Streamlit UI**.

## üß† Architecture Overview

1. Text documents are converted into embeddings using OpenAI
2. Embeddings are stored directly in Oracle Database using a `VECTOR` column
3. Similarity search is performed fully inside Oracle using SQL
4. Results are consumed by:
   - a Jupyter Notebook (cookbook walkthrough)
   - a Streamlit web application (interactive demo)

---

## üóÑÔ∏è Database Schema

The `documents` table stores both the original text and its vector embedding:

```sql
CREATE TABLE documents (
    id NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    content CLOB,
    embedding VECTOR(1536, FLOAT32, DENSE)
);
```
