# Plugin Documentation

This folder contains all Colppy reference documentation bundled with the plugin. The AI uses these when answering questions or running commands.

**Use the `docs-reference` skill** to know which doc to read for each topic (HubSpot, Mixpanel, MRR, ICP, etc.).

Files are the same as in `tools/docs/` at the repo root — copied here so the plugin is self-contained when uploaded to Claude or used in Cursor.
